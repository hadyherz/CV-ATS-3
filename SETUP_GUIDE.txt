╔══════════════════════════════════════════════════════════╗
║           CVForge — Netlify Setup Guide                  ║
║        Follow these steps to go live                     ║
╚══════════════════════════════════════════════════════════╝

Everything runs on Netlify — no separate server needed.
This should take you about 30–45 minutes total.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1 — CREATE YOUR PAYMOB ACCOUNT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to: https://accept.paymob.com/portal2/en/signup
2. Sign up — you can register as an individual (sole trader)
3. Upload your National ID when asked
4. Wait for approval email (usually 1–3 business days)

Paymob accepts: Visa, Mastercard, Meeza, Vodafone Cash, Orange Money
Fee per transaction: ~2.75%
Payouts go directly to your Egyptian bank account


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2 — GET YOUR 3 SECRET KEYS FROM PAYMOB
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Once approved, log in to your Paymob dashboard and collect these:

  KEY 1 — API Key
    → Settings → Account Info → copy "API Key"

  KEY 2 — Integration ID
    → Developers → Payment Integrations
    → Find or create a "Card Payments" integration
    → Copy the Integration ID number

  KEY 3 — HMAC Secret
    → Settings → Account Info → click "View" next to HMAC
    → Copy that value

Save all 3 somewhere safe. You'll paste them into Netlify in Step 4.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3 — DEPLOY TO NETLIFY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to https://netlify.com and create a free account

2. From your Netlify dashboard, click:
   "Add new site" → "Deploy manually"

3. Drag and drop the entire "cvforge-netlify" FOLDER into the upload box
   (the folder containing index.html, netlify.toml, and the netlify folder)

4. Netlify will give you a URL like: https://random-name-123.netlify.app
   → You can rename this later in: Site Settings → General → Site name

That's it — your site is live!


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 4 — ADD YOUR SECRET KEYS TO NETLIFY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

In your Netlify dashboard:
  → Site Configuration → Environment Variables → Add a variable

Add these one by one:

  Variable Name           Value
  ─────────────────────────────────────────────────────
  PAYMOB_API_KEY          [paste your API Key from Step 2]
  PAYMOB_INTEGRATION_ID   [paste your Integration ID]
  PAYMOB_HMAC             [paste your HMAC secret]
  BASE_URL                https://your-site.netlify.app
  PRICE_PIASTERS          1499   (= 14.99 EGP — change as needed)
  ─────────────────────────────────────────────────────

After adding all variables:
  → Deploys → Trigger deploy → Deploy site
  (This restarts the site so it picks up your new keys)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 5 — SET CALLBACK URL IN PAYMOB
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

This tells Paymob where to send your customers after paying.

In Paymob dashboard:
  → Developers → Payment Integrations → click your integration
  → Set "Transaction Response URL" to:
     https://YOUR-SITE.netlify.app/payment-callback

  → Set "Server Callback URL" (webhook) to:
     https://YOUR-SITE.netlify.app/paymob-webhook

Replace YOUR-SITE with your actual Netlify site name.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 6 — TEST WITH PAYMOB TEST CARDS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Before charging real customers, test it with these fake card numbers:

  Successful payment:
    Card: 4987 6542 3456 5888
    Expiry: 05/21  |  CVV: 100

  Failed payment (to test error handling):
    Card: 4000 0000 0000 0002

In Paymob → top menu → switch to "Test" mode first
Switch back to "Live" when you're ready to accept real payments.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
HOW TO UPDATE THINGS IN THE FUTURE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CHANGE THE PRICE:
  1. Netlify → Site Configuration → Environment Variables
     → Update PRICE_PIASTERS (EGP × 100, e.g. 4999 = 49.99 EGP)
  2. Open index.html → find "14.99 EGP" → change to your new price
  3. Re-upload the folder to Netlify

CHANGE FEATURES LIST ON PAYWALL:
  → Open index.html
  → Find "✏️ Edit features list here"
  → Add or remove <li> lines

CHANGE BRAND NAME:
  → Open index.html
  → Find and replace "CVForge" with your name

CHANGE AI INSTRUCTIONS:
  → Open index.html
  → Find "const AI_SYSTEM ="
  → Edit the text between the backticks

RE-DEPLOY AFTER CHANGES:
  → Drag the updated folder onto Netlify again
  → Netlify automatically replaces old files
